# 반복 출력하기

## 문제 설명
`12345^&*()_`를 입력으로 주어지는 $N$번 출력합니다. 줄 수를 받는 함수를 써보라고 하네요.

## 코드

```cpp

#include <iostream>

using namespace std;

int N;

void print_dummy(int n) {
    for (int i = 0; i < n; i++) {
        cout << "12345^&*()_" << '\n';
    }
}

int main() {
    cin >> N;

    print_dummy(N);

    return 0;
}

```

# 별 찍는 것을 5번 반복하기

## 문제 설명

입력도 없이 그냥 `**********` 를 5번 찍으면 됩니다. 함수를 5번 호출해보라네요.


## 코드

```cpp

#include <iostream>
using namespace std;

void print_star() {
    cout << "**********" << '\n';
}

int main() {
    print_star();
    print_star();
    print_star();
    print_star();
    print_star();
    return 0;
}

```

# 숫자로 이루어진 사각형

## 문제 설명

정수 $N$이 주어지면 $N \times N$ 크기의 사각형을 출력하는데, 사각형 안의 숫자들이 1 ~ 9까지로 제한됩니다.

### 입력

```
4
```

### 출력

```
1 2 3 4
5 6 7 8
9 1 2 3
4 5 6 7
```

## 코드

```cpp

#include <iostream>

using namespace std;

int N;

int main() {
    cin >> N;

    int num = 1;

    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++) {
            cout << num << ' ';
            if (num < 9) {
                num++;
            } else {
                num = 1;
            }
        }
        cout << '\n';
    }

    return 0;
}

```

# 출력결과 80

## 문제 설명

```
*****
** **
* * *
** **
*****
```
와 같은 패턴을 출력하기 위한 함수를 `draw` 라고 구현하고 `draw(5)`를 호출하는 상황에서 `(A)`에 알맞은 코드를 고르는 5지선다 문제입니다.

```cpp

void draw(int n) {  
    for (int i=1; i<=n; i++) {    
        for (int j=1; j<=n; j++)      
            if (i==1 || i==n || j==1 || (A) || j == n) 
                cout << '*';
            else 
                cout << ' ';
        cout << '\n';
    }
}

```

## 풀이

이 문제는 패턴을 보고 때려맞추는 문제라고 생각했습니다. 그래서 패턴을 유심히 보고 생각해봤더니 사각형의 모든 경계(border) 부분이 칠해져있고, 가운데는 X자 모양이라고 유추할 수 있었습니다.  

경계 부분을 칠하는 코드는 이미 문제의 if 조건문의 조건으로 주어졌고, `(A)` 부분이 경계를 제외한 안쪽을 칠하는 조건이라고 판단했습니다.  

안쪽에서 X자를 그리기 위해서는 어떻게 할지 고민해보면, (2, 2), (3, 3), (4, 4)와 같이, **대각성분**이 칠해지는 것을 알 수 있습니다. 대각성분을 반복문 인덱스로 표현하면 `i==j`가 됩니다. 이 식으로 하여금 X자의 왼쪽(\\) 부분을 완성할 수 있습니다.  

그 후 오른쪽(/) 부분을 완성하기 위해 고민해보면, (2, 4), (4, 2)와 같이 **`i` 인덱스와 `j` 인덱스의 합이 `n`으로 표현**되는 것을 알 수 있습니다. 즉 `i + j == n + 1`이 됩니다.  

정답은 `j == i or j == n - i + 1` 이었습니다만, `i`를 이항하면 제가 만든 식이 나오는 것을 알 수 있습니다.

# 함수를 이용해 직사각형 만들기

## 문제 설명

$n$과 $m$을 받고 $n \times m$ 크기의 `1`이라는 글자로 만들어진 사각형을 출력하는 문제입니다.

```cpp

#include <iostream>

using namespace std;

int n, m;

void printCol(int col) {
    for (int i = 0; i < col; i++) {
        cout << "1";
    }
}

int main() {
    cin >> n >> m;

    for (int i = 0; i < n; i++) {
        printCol(m);
        cout << '\n';
    }

    return 0;
}

```


# 하고 싶은 말

용두사미(龍頭蛇尾)라는 말이 있습니다. 시작은 창대하나 끝이 빈약한 상황을 일컫는 말입니다. 첫 포스팅이지만, 너무 헤비하게 쓰면 이걸 尾까지 끌고 가지 못할 것이고, 꾸준하게 하지 않는다면 尾를 보지도 못하는 꼴이 됩니다. 결국 **조금씩 천천히라도 꾸준히**가 제일 중요한 것 같습니다.
