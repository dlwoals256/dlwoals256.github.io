# 재귀함수를 이용한 피보나치 수

## 문제 설명

정수 $N$ 이 주어지면 $N$ 번째 피보나치 수를 구합니다.

## 코드

```cpp

#include <iostream>

using namespace std;

int N;

int fib(int n) {
    if (n == 0) return 0;
    if (n == 1 || n == 2) return 1;
    return fib(n - 1) + fib(n - 2);
}

int main() {
    cin >> N;

    cout << fib(N);

    return 0;
}

```

# 출력결과 9

## 문제 설명

`trif(8, 4)`의 출력 결과를 묻습니다. 5지선다입니다.

```cpp

int trif(int n, int k) {
    if (n <= k) return n;
    return trif(n - 1, k) + trif(n - 2, k) + trif(n - 3, k);
}

```

## 풀이

# 홀수 짝수에 따른 출력값

## 문제 설명

정수 $N$ 이 주어지고 $N$ 이 홀수면 $1$ 부터 $N$ 까지의 홀수의 합을, $N$ 이 짝수면 $2$ 부터 $N$ 까지의 짝수의 합을 출력합니다.

## 코드

```cpp

#include <iostream>

using namespace std;

int N;

int sums(int n) {
    if (n < 0) return 0;
    return sums(n - 2) + n;
}

int main() {
    cin >> N;

    cout << sums(N);

    return 0;
}

```
