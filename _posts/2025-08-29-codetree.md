# 출력결과 82

## 문제 설명

`f(4, 3)` 의 값을 묻습니다. 5지선다입니다.

```cpp

int f(int a, int b) {
    if (a < b || b == 0) return 1;
    return f(a - 2, b - 1) + f(a - 1, b) * b;
}

```

## 풀이

재귀 트리를 그려서 풀 수 있습니다. 정답은 15입니다.

# 출력결과 82

## 문제 설명

보기 중에 가장 큰 값을 묻습니다. 5지선다입니다. 보기는:

1. `f(21, 34)`
2. `f(123, 12)`
3. `f(2021, 4)`
4. `f(107, 36)`
5. `f(66, 60)`

와 같습니다.

```cpp

int f(int a, int b) {
    if (a < b || b == 0) return 1;
    return f(a - 2, b - 1) + f(a - 1, b) * b;
}

```

## 풀이

직접 일일이 다 대입해보면 풀 수 있습니다. 정답은 `f(21, 34)`입니다.
